Down below is a list of things that might be useful to know about the website and the geographical map.


  • First register at 'register' tab with a username and password then press the 'login' tab to login.

  • To use the map functionality, login using a username and password.

  • The graphical map can be used in varries ways.

  • Pressing 'Action' displays nodes graphically within the map and also creates a way to add new nodes to specific maps.
    • By pressing 'Action' without any input displays ALL added location nodes in the database (sharing is caring).
    • By pressing 'Action' with only 'name' and then a known map u have added, displays only the nodes for that map (can only be displayed if owned).
    • By pressing 'Action' with all three input fields will add a graphical node location to a specific map that is private (private maps meaning only you can add to it).

  • Pressing 'History' display added nodes for all private maps via text below the geographical map. Displayed with timestamps, map name and the coordinate of the nodes.
    • By pressing 'History' without any input displays all added nodes over all maps (only private once).
    • By pressing 'History' with input name display the added nodes over that specific map (only private once).

  • By double clicking on the graphical map, a pin-point marker will appear (this marker will not be inserted) into any map.
    • By clicking shows the info about the pin-point.
    • By double clicking on these pin-points removes them.
